Adds a customizable marquee ticker to your WordPress site.

== Description ==

This plugin adds a customizable marquee ticker to your WordPress site. You can:

* Enable/disable the marquee
* Display it above your header
* Use a shortcode to display it anywhere
* Customize colors, text, and speed

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-marquee-ticker` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > Marquee Ticker to configure

== Frequently Asked Questions ==

= How do I display the marquee? =

You can either enable it in the settings to show above your header, or use the [marquee_ticker] shortcode anywhere in your content.

= Can I customize the appearance? =

Yes! Go to Settings > Marquee Ticker to change colors, text size, and scrolling speed.
